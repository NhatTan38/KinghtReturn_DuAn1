# Knight Return – 2D Game (Unity)

## Mô tả
Dự án game 2D được phát triển bằng Unity và C#.  
Game gồm hệ thống nhân vật, quái và boss với cơ chế chiến đấu cơ bản.

## Chức năng chính
- Điều khiển nhân vật di chuyển và tấn công
- Hệ thống máu nhân vật và boss
- Xử lý va chạm (Collision)
- Quản lý màn chơi

## Công nghệ sử dụng
- Unity Engine
- C#
- OOP (Object-Oriented Programming)
- Git/GitHub

## Vai trò
- Tự thiết kế và phát triển toàn bộ gameplay
- Xây dựng logic boss và cơ chế chiến đấu
